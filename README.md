# Customer Churn Analysis Dashboard
Power BI dashboard analyzing telecom customer churn using the IBM Telco dataset.

## Pages
- Churn Overview
- Tenure and Trends
- Product Segmentation
- Customer Comparison
- Churn Decomposition Tree
- Geographic Analysis (California)

## Key Findings
- Month-to-month contracts have the highest churn rate (43%)
- Fiber optic customers churn more than DSL users
- Electronic check payment method correlates with highest churn
- Senior customers churn at nearly double the rate of non-seniors
- Competitor-related reasons are the #1 cause of churn

## Tools Used
Power BI Online, Excel, DAX
